Create navigational links within a web site

There are several techniques to nivigate within a web site. The semantic tag for navigation is the <nav>, tag. You can navigate through your web site by using relative addressing. For our examples we will use the unordered list tag (<ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page (on page navigation)

navigational menus are placed in your <header> semantic area. They may also be places in your <footer> area as a sitemap. The nav tag like other semantic tags should have a <hx> tag. HOWEVER this header is optional. If you omit the header tag you will recieve a document outliner error: Untitled section. Until you are shown how to avoiod this error message, it will be acceptable on your document outline.
